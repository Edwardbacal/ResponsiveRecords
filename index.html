Responsive records is a web project in which I, Edward Bacal, recreate some of my favourite album covers in the context of a responsive web browser. This project uses HTML, CSS3, and JavaScript to explore the possibilities and limitations that come with reinterpreting static, fixed-frame images as dynamic, interactive layouts. (Of course, this page only makes sense if you're on a desktop browser).
